# Fraudulent Job Detection System

A Machine Learning based web application that detects whether a job advertisement is Real or Fake using the Extra Trees Classifier, NLP techniques, fraud pattern analysis, and job recommendation features.

## 📌 Project Overview

This project helps job seekers identify fraudulent job advertisements posted on online recruitment platforms. The system analyzes job descriptions, salary details, company information, and suspicious keywords to classify job postings as genuine or fake.

In addition to fraud detection, the system also provides job recommendations based on user interests and job categories, helping users discover safe and relevant job opportunities.

The application uses:
- Natural Language Processing (NLP)
- TF-IDF Feature Extraction
- Extra Trees Classifier
- Fraud Pattern Analysis
- Job Recommendation System

## 📊 Working Process

1. User enters job details  
2. Data preprocessing is performed  
3. TF-IDF extracts important features  
4. Fraud indicators are analyzed  
5. Extra Trees Classifier predicts result  
6. System displays:
   - Real Job
   - Fake Job
7. Recommended jobs are shown to users based on preferences and categories
